DECtalk API

MSS provides an online API to generate DECtalk sounds without the need of Microsoft Windows (or compatible) software.
This service is currently being provided by a Windows Server 2008 R2 Virtual Machine with 4GB of allocated RAM.

Routes

GET /api/list Get the list of DECtalk sounds
Returns JSON array:

[
	{
		"name": "Name of the DECtalk Sound",
		"artist": "Name of whoever originally made the sound",
		"author": "Name of whoever made the DECtalk sound",
		"dectalk": "DECtalk input",
		"html": "HTML that is displayed on the real list",
	}
]

GET /api/gen Generate a DECtalk sound.
Example: GET /api/gen?dectalk=oh%20no
Returns WAV file

POST /api/gen Generate a DECtalk sound.
Send form-data or json with dectalk set to something.
Returns WAV file

GET /dec/[UUID].wav Get the pregenerated sound file for a specific ID
Returns WAV file
