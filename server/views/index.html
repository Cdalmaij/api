API

You can connect your services to DECtalk

This service is currently provided by a Windows 7 virtual machine with 1GB of RAM.

POST

Send a POST request to /api/gen and send a JSON encoded string with your query as input:

{"dectalk": "[dectalk]"}

Maximum Characters: 10,000

GET

Send a GET request to /api/gen and append your query to the URL:

/api/gen?dectalk=[dectalk]

Maximum Characters: 2,000
